Bienvenido al Sistema de Gestion BF

Nuestro sistema de gestión del instituto está diseñado para simplificar y optimizar todas las tareas administrativas y académicas. Con esta herramienta, los usuarios pueden gestionar de manera eficiente la información clave del instituto. El sistema permite visualizar, agregar, editar y eliminar registros directamente desde la base de datos, asegurando que toda la información esté siempre actualizada y sea fácilmente accesible.

Ya sea para administrar datos de estudiantes, profesores, cursos o cualquier otro recurso institucional, nuestra plataforma ofrece una solución integral que facilita la toma de decisiones y mejora la organización dentro del instituto. ¡Empiece a explorar todas las funcionalidades que ofrecemos para llevar la gestión de su institución al siguiente nivel!

Atención Alumnos! Trámites varios

A partir del mes de junio podrá solicitar trámites que se realizan en el Departamento Alumnos por el Sistema de Autogestión. Ej: Certificado de Alumno Regular.

Cuando ingresa con el legajo y la clave tiene que marcar Trámites alumnos y le llegará un código a su correo para luego poder acceder a las distintas opciones de trámites.
